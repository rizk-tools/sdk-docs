## Running Examples

All examples in the documentation are designed to be copy-paste ready. To run them:

1. **Install Rizk SDK:**
   ```bash
   pip install rizk
   ```

2. **Set your API key:**
   ```bash
   export RIZK_API_KEY="your-api-key-here"
   ```

3. **Copy and run any example from the documentation**
